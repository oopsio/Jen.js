Powerful Features

Everything you need for modern web development

Lightning Fast

Static Site Generation by default. Pre-render pages at build time for instant page loads.

TypeScript First

Built with TypeScript strict mode. Full type safety out of the box. Better DX, fewer bugs.

File-Based Routing

Intuitive routing based on your file structure. No complex configuration needed.

Islands Architecture

Partial hydration. Only interactive components become JavaScript. Everything else is static HTML.

Plugin System

Extensible architecture. Build custom plugins to enhance your applications.

Built-in Database

Support for SQLite, MySQL, PostgreSQL, and a built-in JSON database.
