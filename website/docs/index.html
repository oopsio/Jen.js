Introduction to Jen.js

Jen.js is a TypeScript-first web framework for building fast, modern web applications with static site generation (SSG) and server-side rendering (SSR).

What is Jen.js?

Jen.js combines:

  • File-based routing - Routes are defined by your file structure
  • Multiple rendering modes - SSG, SSR, ISR, and PPR
  • Islands architecture - Only hydrate what you need
  • TypeScript first - Full type safety out of the box
  • Built-in features - Database, API routes, authentication, caching, and more
